API Calls
GET
/auth/me 🔒On mount (AuthContext) — load user identity
POST
/circuits/applyGate openEvery step-by-step Next click — body: qubitStates + gate + index
POST
/circuits/run openOn "Run simulation" — body: circuit matrix + shots
POST
/circuits 🔒On "Save" (new circuit)
PUT
/circuits/:id 🔒On "Save" (existing circuit)
User Interactions
Select gate from palette → click cell to place
Ctrl+Z / Cmd+Z → undo last gate
Click placed gate → remove it
Select shots preset → Run simulation
Step-by-step: Start → Next (↑ per step) → Reset
Auto-play toggle → timed step execution
Templates button → navigate to /templates (passes circuit draft)
Save → opens SaveModal (name input) → POST or PUT
Reset → clears circuit + results + live state
View Full Results → navigate to /results
Template load → ConfirmModal if circuit not empty
Client State (hooks)
useCircuitGates — circuit matrix, gate palette
useLiveState — qubitStates[], measurementLog
useFeatureAMultiRun — shots, results, runStatus
useFeatureDStepByStep — queue, status, index
useStepByStepHandlers — start/next/reset callbacks
useUndoStack — undo history
useSaveCircuit — save modal, name draft
useTemplateLoader — pendingLoad / confirm / cancel

User Interactions

Filter input → client-side tag filter (no extra API call)

Use → navigate to /circuit-builder (state: templateToApply)
✏ Edit → open SaveTemplateModal pre-filled
🗑 Delete → ConfirmModal → DELETE /templates/:id
💾 Save circuit → open SaveTemplateModal (new)
"Mine" tab without auth → redirect to /login
